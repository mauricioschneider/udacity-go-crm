# udacity-go-crm

Code for Udacity's Go course final project.

## Dependencies

Run `go get .`

Dependencies are listed in `go.mod`:

```
	github.com/google/uuid v1.5.0
	github.com/gorilla/mux v1.8.1
```

## Starting the server

Run `go run .`

## Testing

Run `go test`

## API documentation

Once you start the server, visit `http://localhost:8000`
